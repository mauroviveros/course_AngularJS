## 📟 Setup

```bash
$ npm install
$ npm start
```

> ### 🛠 Setup backend
> - generate file `./Backend/.env`
> ```
> PORT = <number | default: 4000 >
> MONGODB_SRV = <string | mongoDB URI>
> ```
> - commands on `./Backend`
> ```bash
> $ npm install
> $ npm start
> ```
